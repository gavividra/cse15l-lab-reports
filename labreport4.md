```
$git clone https://github.com/ucsd-cse15l-s23/lab7

```


```
$vim ListExamples.java
```

```
$bash test.sh
```

**Commit and push the resulting change to your Github account (you can pick any commit message!)**
```
$git add *.java

```
